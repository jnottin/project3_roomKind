# project_3_roomKind

project_3_roomKind

## Description

roomKind is a web application intended to galvanizing everyday people to do their part to alleviate homelessness. This applicationa allows for homeless shelters to list empty beds online for those in need, and allows users to do their part by listing thier spare rooms as well, all for free!

## List of features/User Stories

- Shelters can list
- available properties will be shown on homepage
- beds/rooms can be created, updated, or deleted at any time
- Shelters can list empty beds daily
- Residents can list their spare room
  and how many days the rooms are available
- Guests can reserve a bed/room
